# Carpeta de Traducciones

Esta carpeta contiene los archivos de traducción del plugin AI Auto Blog.

## Archivos de Traducción

Los archivos `.po` y `.mo` para diferentes idiomas se colocarán aquí.

### Formato de Nombres

- `ai-auto-blog-es_ES.po` - Español (España)
- `ai-auto-blog-es_ES.mo` - Español compilado
- `ai-auto-blog-en_US.po` - Inglés (EE.UU.)
- `ai-auto-blog-en_US.mo` - Inglés compilado

## Generar Traducciones

Para generar el archivo `.pot`:

```bash
wp i18n make-pot . languages/ai-auto-blog.pot
```

## Text Domain

El text domain del plugin es: `ai-auto-blog`
